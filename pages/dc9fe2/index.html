List-Map过滤

# 根据Map的key过滤掉重复的

	    Map<String, String> map1 = new HashMap<>();
        Map<String, String> map2 = new HashMap<>();
        Map<String, String> map3 = new HashMap<>();

        map1.put("id", "a1");
        map1.put("name", "shigure");
        map1.put("age", "22");

        map2.put("id", "a1");
        map2.put("name", "shigure");
        map2.put("age", "32");

        map3.put("id", "a2");
        map3.put("name", "test");
        map3.put("age", "32");

        List<Map<String, String>> list = Arrays.asList(map1, map2, map3);

        System.out.println("去重前："+list);

        //结果放在list中
        List<Map<String, String>> distinctList = list.stream()
                .collect(Collectors.collectingAndThen(
                        Collectors.toCollection(() ->
                                new TreeSet<>(Comparator.comparing(value -> value.get("id") + ";" + value.get("name")))),
                        ArrayList::new));
        System.out.println("去重后List："+distinctList);
//结果放在set中
        Set<Map<String, String>> set = list.stream()
                .collect(Collectors.toCollection(() ->
                        new TreeSet<>(Comparator.comparing(value -> value.get("id") + ";" + value.get("name")))));
        System.out.println("去重后Set："+set);

结果输出：

输出：：：
去重前：[{name=shigure, id=a1, age=22}, {name=shigure, id=a1, age=32}, {name=test, id=a2, age=32}]
去重后List：[{name=shigure, id=a1, age=22}, {name=test, id=a2, age=32}]
去重后Set：[{name=shigure, id=a1, age=22}, {name=test, id=a2, age=32}]
